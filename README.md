# chzzk-vod-dl

cld : Chzzk VOD Downloader

```bash
$ cvd
cvd [Chzzk VOD Downloader]
Usage:
  cvd list <channel id>
  cvd info <video no>
  cvd download <video no>
  cvd all <channel id>
```
